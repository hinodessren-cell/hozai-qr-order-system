# 補材 QR 発注管理システム

PCとスマートフォンの両方で使える、補材のQR発注・入荷管理システムです。現場のQR看板から対象品目を開き、発注から入荷、完了までの状況を共有できます。

## 主な機能

- 品目ごとの発注用QRコードとA4印刷用QR看板
- QRから品目を直接開いて発注
- 発注待ち、入荷待ち、完了の進捗管理
- 発注履歴と品目検索
- PC・スマートフォン対応のレスポンシブ画面
- 色、表示密度、カード列数、表示項目、状態名、既定数量、看板サイズなどの詳細設定
- 旧システムの品目1,512件と発注履歴7件を移行するデータ

## 技術構成

- Next.js / React（Vinext）
- Cloudflare D1 / Drizzle ORM
- Sitesによるインターネット公開

## 開発環境での起動

Node.js 22.13以上を用意して、次のコマンドを実行します。

```bash
npm install
npm run dev
```

ビルド確認:

```bash
npm run build
```

## データベース

- `drizzle/0000_initial.sql`: テーブル作成
- `drizzle/0001_legacy_data.sql`: 旧システムから移行した初期データ
- `.openai/hosting.json`: 公開環境のD1バインディング設定

## 主な画面

- 概要: 件数と進行中の発注を表示
- QR読取: QR看板から品目を選択
- 発注管理: 発注待ち・入荷待ちを更新
- 履歴: 完了を含む発注履歴
- QR看板: 品目別QRを配置して印刷
- 品目: 品目マスターの閲覧と発注
- 詳細設定: レイアウト、表示、発注フロー、印刷設定を変更
